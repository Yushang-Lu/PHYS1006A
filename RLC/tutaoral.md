# `rlc.tex` 与 `process.py` 食用指南

`RLC/rlc.tex` 是这次 RLC 暂态电路实验的半成品，使用者需要自己记录数据

`RLC/process.py` 用来实现实验数据的处理。脚本中已经写入了示例数据：

- `t_us`：各个峰值对应的时间（单位 `us`）
- `uc_v`：各个峰值电压（单位 `V`）
- `e_v`：信号源电压 `E`

如果你要处理自己的实验数据，只需要修改这三个量，然后在项目根目录运行：

```bash
python3 RLC/process.py
```

脚本会输出：

- `ln(U_C - E)` 的计算结果
- 线性拟合得到的 `k0` 和 `k1`
- `1/tau` 和 `tau`

然后把这些结果填写回你的实验报告即可。需要注意的是，脚本依赖 `numpy`；如果运行时提示缺少该模块，需要先在当前 Python 环境中安装它。另外，这个脚本采用的是 `ln(U_C - E)`，适用于报告中欠阻尼充电时 `U_C > E` 的那组峰值数据。
