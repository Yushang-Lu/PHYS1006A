# 用惠斯通电桥测电阻实验

## `wsb.tex` 食用说明

本目录包含一份大学物理实验报告 LaTeX 模版，用于整理“用惠斯通电桥测电阻”实验的原始数据、现象分析和讨论题答案。

目前文件分工如下：

- `wsb.tex`：实验报告正文模版
- `wsb.pdf`：已编译出的实验报告 PDF
- `tutorial.md`：本说明文档

建议的使用顺序是：

1. 先在实验中记录原始数据
2. 将测得的数据手动整理到 `wsb.tex` 的表格中
3. 根据实验现象和数据结果补充“实验结论及现象分析”
4. 完成讨论题答案
5. 编译 `wsb.tex` 生成实验报告 PDF

## 2. `wsb.tex` 的作用与用法

`wsb.tex` 是实验报告的主文件。

目前它已经完成了这些工作：

- 设置了报告版式和页眉页脚
- 预留了“实验目的”“实验预习”“实验现象及数据记录”“实验结论及现象分析”“讨论问题”等部分
- 提供了两张原始数据记录表格
- 给出了实验结论和讨论题的一版示例文字

### 2.1 你通常需要修改哪些部分

建议重点检查和修改：

- 页首的班级、学号、姓名、实验日期
- `\section{实验目的}` 的内容
- `\section{实验预习}` 中的惠斯通电桥电路图和平衡条件说明
- `\section{实验现象及数据记录}` 中两张表格的具体数据
- `\section{实验结论及现象分析}` 中关于灵敏度变化的分析
- `\section{讨论问题}` 中的答案表述

特别注意：

- `wsb.tex` 当前的很多文字是示例答案，不一定与你的实测数据完全对应
- 如果你实际测得的灵敏度变化趋势与当前模板不一致，需要把分析结论一起改掉
- 表格中的空白栏位都需要根据你的实验结果手动填写

### 2.2 原始数据记录部分怎么填

当前 `wsb.tex` 中预留了两张表：

- “惠斯通电桥测量电阻”
- “惠斯通电桥灵敏度测量(1 `k\Omega`)”

其中常见量的含义大致如下：

- `N`：桥臂比值
- `R_s`：标准电阻
- `R_x`：待测电阻
- `\Delta R_s`：标准电阻的改变量
- `\Delta n`：检流计偏转格数
- `S`：灵敏度

整理数据时建议注意：

- 单位要与表头保持一致
- 数值位数尽量统一，便于排版
- 如果某些量需要现场计算，建议先在纸上或计算器中确认后再填入 LaTeX 表格

### 2.3 当前模板里已有的分析内容

`wsb.tex` 目前已经写了一版关于“不同 `N` 值下灵敏度变化”的示例分析，以及 3 个讨论题答案。

这些内容可以直接作为参考，但建议你至少检查以下几点：

- “当 `N = 1` 时灵敏度最高”是否与你的实验现象一致
- 对灵敏度影响因素的分析是否需要增删
- 讨论题中的表述是否符合老师要求的书写风格

## 5. 常见问题

### 5.1 LaTeX 编译失败

优先检查：

- 是否使用了 `xelatex`
- `wsb.tex` 中是否有未配对的括号、公式符号或环境
- 表格中是否误写了特殊字符

### 5.2 表格内容太长，导致排版不好看

优先检查：

- 单元格里是否写入了过长的说明文字
- 数值位数是否过多
- 是否把本应写在正文里的分析误放进了表格

### 5.3 模板里的分析和我的实验结果不一致

这是正常现象，因为 `wsb.tex` 当前更像是一份参考模板。你应当根据自己的实验数据修改：

- 灵敏度大小规律
- 误差来源分析
- 讨论题答案中的具体表述

### 5.4 LaTeX 编译出现 Warning，但最终生成了 PDF

如果只是 Warning 且 PDF 已正常输出，一般可以先忽略。优先检查表格、公式和中文显示是否正常即可。
